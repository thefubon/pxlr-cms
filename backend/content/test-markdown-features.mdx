## Форматирование текста

***Жирный и курсивный одновременно*** тоже должен работать.

## Inline код

Вот пример inline кода: `const variable = "value"` и `/package.json` файл.

Еще примеры: `npm install`, `git commit`, `docker build`.

## Блоки кода

Простой блок кода:

```
function hello() {
  console.log("Hello, World!");
}
```

Блок кода с языком:

```javascript
const express = require('express');
const app = express();

app.get('/', (req, res) => {
  res.send('Hello World!');
});

app.listen(3000, () => {
  console.log('Server running on port 3000');
});
```

Bash команды:

```bash
npm install marked
npm run dev
git add .
git commit -m "fix markdown"
```

## Списки
