# 📝 CHANGELOG - PXLR CMS Admin Panel

Журнал изменений админ-панели CMS с подробным описанием всех улучшений и новых функций.

---

## 🎉 **v0.0.1-alfa** *(17.07.2025)* 🚀

### 🔥 **Основные изменения**

#### 1. **Полный рефакторинг системы Preview**
- ✅ **Компоненты теперь отображаются как настоящие** - вместо карточек с информацией
- ✅ **Поддержка всех компонентов**: Hero, Card, CallToAction, FeatureGrid, Quote, Accordion, Tabs, ImageGallery
- ✅ **Продакшен-подобный рендеринг** с правильными стилями и градиентами
- ✅ **SVG иконки и изображения** для полного соответствия фронтенду

#### 2. **Полная поддержка Tailwind CSS**
- ✅ **200+ Tailwind классов** - bg-*, text-*, p-*, m-*, w-*, h-*, flex, grid и др.
- ✅ **Конвертация в inline-стили** для корректного отображения в preview
- ✅ **Поддержка className и class** атрибутов
- ✅ **Сохранение существующих style** атрибутов

#### 3. **Интерактивные компоненты**
- ✅ **Функциональные аккордеоны** - клик для открытия/закрытия с анимацией
- ✅ **Рабочие табы** - переключение между вкладками
- ✅ **Модальные окна для галерей** - полноэкранный просмотр изображений
- ✅ **Плавные анимации** с CSS transitions

#### 4. **Улучшенный редактор кода**
- ✅ **Подсветка синтаксиса** в реальном времени:
  - 🟣 MDX компоненты
  - 🟢 Атрибуты
  - 🔴 Строки
  - 🔵 Markdown заголовки
  - 🟡 Inline код
- ✅ **Автосохранение** с визуальными индикаторами
- ✅ **Горячие клавиши**:
  - `Ctrl+S` - Сохранить
  - `Ctrl+Shift+F` - Форматировать
  - `Ctrl+Shift+C` - Библиотека компонентов
  - `Escape` - Закрыть модальные окна

#### 5. **Система управления датами постов**
- ✅ **Редактирование дат** в формах создания/редактирования
- ✅ **Быстрые действия**:
  - 🕒 "Сейчас" - текущее время
  - ⭐ "В топ" - будущая дата для поднятия в топ
  - 📦 "В архив" - прошлая дата для отправки в низ
  - 🔄 "Авто" - очистка для автоматической даты
- ✅ **Визуальные индикаторы**:
  - 🟢 Зеленая подсветка для будущих дат
  - 📅 Бейджи "ПЕРВЫЙ" и "CUSTOM"
  - ⭐ Иконки для будущих дат
- ✅ **Сортировка по дате** вместо времени изменения файла

#### 6. **Убраны ограничения ширины**
- ✅ **Удален max-w-4xl** со всех страниц редактирования
- ✅ **Полная ширина экрана** для максимального рабочего пространства
- ✅ **Сохранены ограничения** для информационных страниц (главная, настройки)

#### 7. **Переключаемые режимы просмотра**
- ✅ **Режим "Только редактор"** (по умолчанию) - полная ширина для кода
- ✅ **Режим "Live Preview"** - разделенный экран 50/50
- ✅ **Интерактивные кнопки** переключения с hover эффектами
- ✅ **Визуальная обратная связь** - активная кнопка подсвечивается

#### 8. **Документация и руководства**
- ✅ **CHANGELOG.md** - подробный журнал всех изменений
- ✅ **ADMIN_FEATURES.md** - полное руководство по функциям админки
- ✅ **Обновленный README.md** - ссылки на документацию и быстрый доступ
- ✅ **Структурированное описание** всех возможностей
- ✅ **Примеры кода** и инструкции по использованию

#### 9. **Динамическое меню и Drag & Drop**
- ✅ **Динамическое меню** - автоматическое обновление на фронтенде
- ✅ **Drag & Drop сортировка** - перетаскивание пунктов меню
- ✅ **Visual feedback** - placeholder и анимации при перетаскивании
- ✅ **Автосохранение порядка** - обновление порядка в реальном времени
- ✅ **Нумерация пунктов** - визуальные индикаторы порядка

#### 10. **Автоконвертация HTML → React**
- ✅ **Умная конвертация** - автоматическая замена HTML атрибутов на React
- ✅ **30+ атрибутов** - class→className, for→htmlFor, tabindex→tabIndex и др.
- ✅ **Debounced обработка** - конвертация с задержкой для избежания спама
- ✅ **Кнопка переключения** - включение/отключение автоконвертации
- ✅ **Контекстная конвертация** - только внутри HTML тегов
- ✅ **Сохранение позиции курсора** - курсор остается на месте после конвертации

## 🛠️ **Технические улучшения**

### **Backend изменения** (`backend/index.js`)
- Обновлен API `/api/posts/:slug` для поддержки кастомных дат
- Изменена сортировка постов по `frontmatter.date`
- Добавлена валидация и обработка дат при сохранении

### **Frontend изменения** (`backend/public/index.html`)
- Полностью переписана функция `processComponents()`
- Добавлена функция `convertTailwindToInlineStyles()`
- Реализованы интерактивные функции для компонентов
- Добавлена система режимов редактора
- Улучшена система навигации и автосохранения

### **CSS улучшения**
- Новые классы для режимов редактора (`.editor-only`, `.preview-mode`)
- Стили для активных кнопок (`.mode-btn.active`)
- Плавные переходы и анимации
- Адаптивная сетка для компонентов
