## ✏️ **Редактор контента**

### **Режимы просмотра**
- 📝 **Режим "Только редактор"** (по умолчанию):
  - Полная ширина экрана для кода
  - Максимальное рабочее пространство
  - Фокус на написании контента

- 👁️ **Режим "Live Preview"**:

### **Подсветка синтаксиса**
- 🟣 **MDX компоненты** - `<Hero>`, `<Card>` и т.д.
- 🟢 **Атрибуты** - `title="text"`, `className="style"`
- 🔴 **Строки** - текст в кавычках
- 🔵 **Markdown заголовки** - `#`, `##`, `###`
- 🟡 **Inline код** - `` `code` ``

### **Live Preview функции**
- ⚡ **Мгновенное обновление** при изменении кода
- 🎨 **Production-качество рендеринг** компонентов
- 🖱️ **Интерактивные элементы** (аккордеоны, табы, галереи)
- 📱 **Точное соответствие** фронтенду

### **Автосохранение**
- 💾 **Автоматическое сохранение** каждые 2 секунды
- 🟡 **Индикатор "Unsaved"** (янтарный цвет)
- 🟢 **Индикатор "Saved"** (зеленый цвет)

## 🧩 **Библиотека компонентов**

### **Доступные компоненты**

#### **1. Hero - Главный баннер**
```mdx
<Hero
  title="Заголовок"
  subtitle="Подзаголовок"
  buttonText="Кнопка"
  buttonLink="/ссылка"
  backgroundImage="url(image.jpg)"
/>
```
- 🎨 **Градиентный фон** с поддержкой изображений
- 🔘 **CTA кнопка** с настраиваемым текстом и ссылкой
- ⭐ **SVG иконки** для украшения

#### **2. Card - Карточка контента**
```mdx
<Card
  title="Заголовок"
  description="Описание"
  link="/ссылка"
  icon="🚀"
/>
```
- 🎯 **Белый фон** с тенью
- 🔗 **Поддержка ссылок** с hover эффектами
- ➡️ **SVG стрелка** для ссылок

#### **3. CallToAction - Призыв к действию**
```mdx
<CallToAction
  title="Заголовок"
  description="Описание"
  buttonText="Действие"
  buttonLink="/ссылка"
  variant="primary" // или "secondary"
/>
```
- 🎨 **Два варианта**: primary (синий) и secondary (серый)
- 🔘 **Стилизованная кнопка** с анимацией

#### **4. FeatureGrid - Сетка возможностей**
```mdx
<FeatureGrid
  items='[{"title":"Название","description":"Описание","icon":"🔥"}]'
  columns="3"
/>
```
- 📱 **Адаптивная CSS Grid** сетка
- 🎨 **Круглые иконки** с цветным фоном
- 📊 **Настраиваемое количество колонок**

#### **5. Quote - Цитата**
```mdx
<Quote
  text="Текст цитаты"
  author="Автор"
  avatar="avatar.jpg"
/>
```
- 💬 **SVG иконки** кавычек
- 👤 **Поддержка аватаров** автора
- 🎨 **Стильное оформление** текста

#### **6. Accordion - Аккордеон**
```mdx
<Accordion
  items='[{"title":"Вопрос","content":"Ответ"}]'
  allowMultiple="false"
/>
```
- 🖱️ **Интерактивное открытие/закрытие**
- 🔄 **Анимация поворота** стрелок
- ⚙️ **Настройка множественного** открытия

#### **7. Tabs - Вкладки**
```mdx
<Tabs
  items='[{"title":"Вкладка","content":"Содержимое"}]'
/>
```
- 🖱️ **Переключение между вкладками**
- 🎨 **Активная вкладка** выделяется
- 📱 **Адаптивный дизайн**

#### **8. ImageGallery - Галерея изображений**
```mdx
<ImageGallery
  images='["image1.jpg","image2.jpg"]'
  columns="3"
/>
```
- 🖼️ **Модальное окно** для полноэкранного просмотра
- 📱 **Адаптивная сетка** изображений
- 🎨 **Hover эффекты** с масштабированием

### **Поддержка Tailwind CSS**
- 🎨 **200+ классов** - полная поддержка
- 🎯 **Все категории**: colors, spacing, sizing, layout, typography
- 🔄 **Автоматическая конвертация** в inline-стили для preview
- ✅ **Поддержка `className` и `class`** атрибутов

#### **Примеры использования:**
```html
<div className="bg-slate-500 text-white p-6 rounded-lg">
  <h3 className="text-2xl font-bold mb-4">Кастомный блок</h3>
  <p className="text-lg">Любой HTML с Tailwind!</p>
</div>

<div className="flex gap-4 my-8">
  <div className="bg-blue-100 p-4 rounded flex-1">Колонка 1</div>
  <div className="bg-green-100 p-4 rounded flex-1">Колонка 2</div>
</div>
```

## ⚛️ **Автоконвертация HTML → React**

### **Умная замена атрибутов**
Система автоматически конвертирует HTML атрибуты в React эквиваленты во время набора кода.

### **Поддерживаемые атрибуты**
- `class=` → `className=`
- `for=` → `htmlFor=`
- `tabindex=` → `tabIndex=`
- `readonly=` → `readOnly=`
- `maxlength=` → `maxLength=`
- `autofocus=` → `autoFocus=`
- `autoplay=` → `autoPlay=`
- `contenteditable=` → `contentEditable=`
- `crossorigin=` → `crossOrigin=`
- `datetime=` → `dateTime=`
- И **20+ других** атрибутов

### **Особенности работы**
- ⏱️ **Debounced обработка** - конвертация через 500ms после ввода
- 🎯 **Контекстная замена** - только внутри HTML тегов
- 📍 **Сохранение курсора** - позиция курсора остается на месте
- 🔔 **Уведомления** - показ замененных атрибутов
- 🎛️ **Кнопка переключения** - зеленая ⚛️ (включено) / красная ⚛️ (отключено)

### **Пример работы**
```html
Ввод:    <div class="container" for="input">
Результат: <div className="container" htmlFor="input">
```
