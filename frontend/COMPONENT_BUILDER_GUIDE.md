# 🧩 Руководство по конструктору компонентов PXLR CMS

## Обзор

Система конструктора компонентов PXLR CMS позволяет создавать богатый контент с помощью готовых интерактивных блоков без написания кода.

## 🚀 Как использовать

### 1. Основная админка
- Откройте админку: `http://localhost:3333`
- Перейдите к редактированию поста или страницы
- Используйте стандартные возможности

### 2. Улучшенная админка (Enhanced Editor)
- Откройте улучшенную админку: `http://localhost:3333/enhanced-admin.html`
- Полноэкранный редактор с живым предпросмотром
- Встроенный конструктор компонентов

## 📚 Доступные компоненты

### 📐 Макет
- **Hero секция** - Главный баннер с заголовком и кнопкой
- **Сетка возможностей** - Отображение функций в виде сетки

### 📝 Контент  
- **Карточка** - Информационный блок с заголовком и ссылкой
- **Призыв к действию** - Яркий блок для конверсии
- **Цитата** - Оформленная цитата с автором

### ⚡ Интерактив
- **Аккордион** - Раскрывающиеся FAQ
- **Вкладки** - Переключаемый контент

### 🎨 Медиа
- **Галерея изображений** - Сетка картинок с lightbox

## 🔧 Использование в редакторе

### Способ 1: Через конструктор (Enhanced Editor)
1. Откройте улучшенную админку
2. Нажмите кнопку **"🧩 Компоненты"**
3. Выберите категорию или просмотрите все
4. Кликните на нужный компонент
5. Компонент автоматически вставится в текущую позицию курсора

### Способ 2: Ручная вставка
Вставьте код компонента в MDX файл:

```mdx
<Accordion items={[
  {
    title: "Вопрос 1",
    content: "Ответ на вопрос с <strong>HTML</strong> тегами"
  },
  {
    title: "Вопрос 2",
    content: "Второй ответ"
  }
]} allowMultiple={false} />
```

## 📋 Примеры использования

### Hero секция
```mdx
<Hero
  title="Добро пожаловать!"
  subtitle="Создавайте потрясающие страницы"
  buttonText="Начать"
  buttonLink="/get-started"
/>
```

### Карточка контента
```mdx
<Card 
  title="Заголовок"
  content="Описание карточки"
  link="/подробнее"
  linkText="Узнать больше"
/>
```

### Сетка возможностей
```mdx
<FeatureGrid features={[
  {
    icon: "🚀",
    title: "Быстро",
    description: "Молниеносная работа"
  },
  {

    title: "Красиво",

  }

```

### Интерактивные вкладки
```mdx
<Tabs tabs={[
  {
    label: "Первая вкладка",
    content: "<p>Содержимое первой вкладки</p>"
  },
  {
    label: "Вторая вкладка",
    content: "<p>Содержимое второй вкладки</p>"
  }

```

### Аккордион FAQ
```mdx
<Accordion items={[
  {
    title: "Часто задаваемый вопрос?",
    content: "Подробный ответ с возможностью использования HTML тегов"
  }
]} allowMultiple={true} />
```

### Цитата
```mdx
<Quote

  author="Имя Автора"
  position="Должность, Компания"
```

### Призыв к действию

## ✨ Дополнительные возможности

### Форматирование кода
- Кнопка **"✨ Форматировать"** автоматически улучшает разметку
- Удаляет лишние переносы строк
- Выравнивает заголовки

### Живой предпросмотр
- Изменения отображаются в реальном времени справа от редактора
- Подсветка синтаксиса для лучшего восприятия
- Компоненты отображаются как в финальной версии

### Уведомления
- Всплывающие уведомления об успешных действиях
- Индикаторы автосохранения

## 🔧 Настройка

### Добавление новых компонентов
1. Создайте React компонент в `frontend/src/components/PageBuilder/`
2. Добавьте экспорт в `frontend/src/components/PageBuilder/index.tsx`
3. Зарегистрируйте в `COMPONENT_TEMPLATES` 
4. Добавьте в `mdxComponents` объект

### Кастомизация стилей
- Все компоненты используют Tailwind CSS
- Изменяйте стили в соответствующих файлах компонентов

## 🚨 Важные замечания

- Компоненты работают только в MDX файлах
- HTML внутри компонентов должен быть валидным
- Некоторые компоненты требуют изображения (настройте пути)
- В production режиме dev-контент скрывается автоматически
