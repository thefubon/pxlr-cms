# Добро пожаловать в PXLR CMS!

Это ваш первый пост в новой системе управления контентом **PXLR CMS**. Система создана для разработчиков, которые хотят быстро добавить CMS в свои Next.js проекты.

## Что особенного в PXLR CMS?

### 🚀 Простота использования
- Никаких баз данных - все хранится в файлах
- Готов к работе за считанные минуты
- Интуитивно понятный интерфейс админки

### ⚡ Современные технологии
- **Next.js 15** с App Router
- **React 19** с новейшими возможностями
- **TypeScript** для типобезопасности
- **Tailwind CSS 4** для стилизации
- **MDX** для богатого контента

### 📝 Гибкое управление контентом
- Поддержка Markdown разметки
- Автосохранение изменений
- Предварительный просмотр
- Управление меню сайта

## Как начать работу?

1. **Создайте контент** в админке
2. **Настройте меню** для навигации
3. **Опубликуйте посты** для вашей аудитории
4. **Настройте дизайн** под ваши потребности

## Возможности Markdown

В постах вы можете использовать весь функционал Markdown:

- **Жирный текст** и *курсив*
- `Код в строке` и блоки кода
- Списки и нумерация
- [Ссылки](https://github.com) на внешние ресурсы

```javascript
// Пример кода
function welcomeToPlxr() {
  console.log("Добро пожаловать в PXLR CMS!");
}
```

## Что дальше?

- Изучите админку по адресу [localhost:3333](http://localhost:3333)
- Создайте свои первые страницы и посты
- Настройте меню сайта под ваши потребности
- Наслаждайтесь удобным управлением контентом!
