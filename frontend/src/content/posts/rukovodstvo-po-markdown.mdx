Markdown - это простой язык разметки, который позволяет форматировать текст с помощью обычных символов.

## Заголовки

Используйте символы `#` для создания заголовков:

```
# Заголовок 1 уровня
## Заголовок 2 уровня  
### Заголовок 3 уровня
```

## Форматирование текста

**Жирный текст** - используйте `**текст**`
*Курсив* - используйте `*текст*`
`Код` - используйте обратные кавычки

## Списки

Маркированный список:
* Пункт 1
* Пункт 2
* Пункт 3

Нумерованный список:
1. Первый пункт
2. Второй пункт
3. Третий пункт
