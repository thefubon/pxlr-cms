# Руководство по началу работы с PXLR CMS

Это подробное руководство поможет вам быстро освоить все возможности PXLR CMS и начать создавать качественный контент для вашего сайта.

## Шаг 1: Знакомство с интерфейсом

### Dashboard
Главная панель показывает:
- Общую статистику сайта
- Количество страниц и постов
- Размер контента
- Последние изменения

### Навигация
В левой панели доступны разделы:
- **Dashboard** - общая информация
- **Главная страница** - редактирование home page
- **Страницы** - управление статическими страницами
- **Посты** - блог и новости
- **Меню** - настройка навигации
- **Настройки** - конфигурация сайта

## Шаг 2: Создание первой страницы

1. Перейдите в раздел **"Страницы"**
2. Нажмите **"Создать страницу"**
3. Заполните поля:
   - **Заголовок** - название страницы
   - **Slug** - URL страницы (автоматически генерируется)
   - **Содержимое** - текст в формате Markdown

### Пример содержимого:
```markdown
# О нашей компании

Мы занимаемся разработкой **современных веб-решений** для бизнеса.

## Наши услуги:
- Разработка сайтов
- Мобильные приложения
- Консультации по UX/UI

[Связаться с нами](/contacts)
```

## Шаг 3: Написание постов

Посты отличаются от страниц тем, что:
- Имеют дату публикации
- Показываются в хронологическом порядке
- Подходят для новостей и блога

### Структура поста:
1. **Заголовок** - привлекающее внимание название
2. **Описание** - краткая аннотация для SEO
3. **Содержимое** - основной текст с изображениями и форматированием

## Шаг 4: Настройка меню

1. Откройте раздел **"Меню"**
2. Добавьте пункты навигации:
   - **Название** - текст ссылки
   - **URL** - адрес страницы
   - **Порядок** - позиция в меню

### Рекомендуемая структура:
- Главная (/)
- О нас (/about)
- Услуги (/services)
- Блог (/posts)
- Контакты (/contacts)

## Шаг 5: Оптимизация и настройки

### SEO оптимизация:
- Используйте описательные заголовки
- Добавляйте мета-описания
- Оптимизируйте URL (slug)
- Структурируйте контент заголовками

### Производительность:
- Оптимизируйте изображения
- Используйте кеширование
- Минимизируйте размер контента

## Полезные советы

### Markdown разметка:
- `**текст**` для жирного начертания
- `*текст*` для курсива
- `[текст](url)` для ссылок
- `![alt](url)` для изображений

### Горячие клавиши:
- **Ctrl+S** - сохранить изменения
- **Ctrl+Z** - отменить действие
- **Tab** - увеличить отступ

### Автосохранение
Система автоматически сохраняет изменения каждые 2 секунды, поэтому вы не потеряете свою работу.
