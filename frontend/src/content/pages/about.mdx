## Философия проекта

Система разработана с фокусом на **простоту** и **производительность**. Мы верим, что управление контентом не должно быть сложным.

### Ключевые принципы

- **Файловая система** - никаких баз данных, все хранится в файлах
- **Простота развертывания** - готов к работе за считанные минуты
- **Современные технологии** - Next.js 15, React 19, Tailwind CSS
- **Российская локализация** - полностью на русском языке

## Технический стек

- **Frontend**: Next.js 15 + React 19 + TypeScript
- **Backend**: Express.js + Node.js
- **Стили**: Tailwind CSS 4
- **Контент**: MDX + Markdown
- **Хранение**: JSON файлы

## Возможности

### Для редакторов
- Удобный интерфейс редактирования
- Предварительный просмотр
- Автосохранение изменений
- Управление меню сайта

### Для разработчиков  
- Легкая интеграция с Next.js
- Поддержка TypeScript
- Расширяемая архитектура
- Готовые API endpoints

Все изменения сохраняются мгновенно и сразу отображаются на вашем сайте! ✨
